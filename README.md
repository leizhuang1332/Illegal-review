# 视频违规审核系统

基于规则+AI双引擎架构的视频违规审核系统，支持多模态内容分析和智能决策融合。

## 架构设计

系统采用分层架构设计，各模块职责清晰、解耦良好：

```
┌─────────────────────────────────────────────────────────────┐
│                      输入层                                 │
│  [视频文件] [视频URL] [直播流] [视频片段]                    │
└───────────────────┬─────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────────────┐
│                    预处理层                                 │
│  [视频解码] → [音频提取] → [帧采样优化] → [内容识别]         │
└───────────────────┬─────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────────────┐
│                    分析引擎层                               │
│  [图像识别] [音频分析] [文本分析] [规则引擎] [AI引擎]        │
└───────────────────┬─────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────────────┐
│                    融合决策层                               │
│  [规则评估] → [AI推理] → [权重融合] → [置信度校准] → [决策]   │
└───────────────────┬─────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────────────┐
│                      输出层                                 │
│  [审核结果] [违规类型] [置信度] [建议操作] [证据定位]         │
└─────────────────────────────────────────────────────────────┘
```

## 核心特性

- **规则+AI双引擎**：规则引擎处理明确违规模式，AI引擎处理模糊场景
- **多模态分析**：支持图像、音频、文本的综合分析
- **智能采样**：基于场景变化检测的帧采样策略
- **动态权重融合**：根据置信度动态调整决策路径
- **反馈闭环**：人工复核结果回流优化模型和规则

## 技术栈

| 模块 | 技术方案 |
|------|----------|
| 语言 | Python 3.13+ |
| 框架 | FastAPI |
| 视频处理 | FFmpeg + OpenCV |
| 语音识别 | Whisper |
| OCR识别 | EasyOCR |
| 文本处理 | BERT/RoBERTa |
| 物体检测 | YOLOv8 |
| 数据库 | PostgreSQL + Redis |
| 消息队列 | Kafka |

## 快速开始

### 环境要求

- Python 3.13+
- uv (Python包管理器)

### 安装依赖

```bash
uv sync
```

### 运行命令

```bash
# 启动API服务
python -m src.illegal_review.cli server --host localhost --port 9000

# 访问API 文档
访问 `http://localhost:9000/docs#/
```

## 配置说明

配置文件位于 `src/illegal_review/config/settings.py`，包含各模块的配置参数。
